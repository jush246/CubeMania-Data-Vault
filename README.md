# CubeMania-Data-Vault
# 📊 2025 큐브매니아 연간 통계 분석기

## ✨ 주요 기능
- **수능 등급제 도입**: 전체 조회수 상위 4%를 1등급으로 판정하는 등급 시스템.
- **개인 성적표 발행**: 닉네임 입력 시 평균 조회수, 최고 조회수, 화력 순위 요약.
- **명예의 전당**: 2025년 카페 전체 TOP 20 게시글 추출.
- **내 인기글 분석**: 특정 유저가 작성한 글 중 조회수 TOP 20 자동 분류.

## 🛠️ 기술 스택
- **언어**: Python 3.14
- **라이브러리**: Pandas, Openpyxl
- **배포**: PyInstaller (.exe)
